# 企業詳細 画面 詳細設計

---

## 1. 基本情報

| 項目         | 内容                     |
| ------------ | ------------------------ |
| 画面名       | 企業詳細                 |
| 目的         | 企業の詳細情報を確認する |
| 対象スキーマ | Company                  |

---

## 2. ルーティング

| 種別 | パス                          |
| ---- | ----------------------------- |
| 一覧 | `/companies`                  |
| 詳細 | `/companies/[companyId]`      |
| 編集 | `/companies/[companyId]/edit` |

---

## 3. 表示項目

| 項目名 |
| ------ |
| ID     |
| 企業名 |
| 連絡先 |
| 業種   |
| 担当者 |

---

## 4. 機能

- 企業情報の表示
- 編集画面への遷移ボタン表示
- 一覧へ戻るボタン表示

---

## 5. 挙動（画面遷移）

| 操作           | 挙動                                 |
| -------------- | ------------------------------------ |
| 編集ボタン押下 | `/companies/[companyId]/edit` へ遷移 |
| 戻るボタン押下 | `/companies` へ遷移                  |

---

## 6. 権限（MVP）

| 操作 | 権限                                         |
| ---- | -------------------------------------------- |
| 参照 | 全ユーザー可                                 |
| 編集 | ※ 編集画面側で制御（この画面では制御しない） |

---

## 7. API仕様

### 7.1 企業詳細取得

| 項目     | 内容                         |
| -------- | ---------------------------- |
| Method   | GET                          |
| Endpoint | `/api/companies/[companyId]` |
| 用途     | 企業詳細情報の取得           |

#### レスポンス（例）

```ts
{
  id: string;
  name: string;
  industry: string;
  contact: string | null;
  assignee: string;
}
```

#### エラー

| ステータス | 内容           |
| ---------- | -------------- |
| 401        | 未認証         |
| 404        | 存在しない企業 |
| 500        | 想定外エラー   |
